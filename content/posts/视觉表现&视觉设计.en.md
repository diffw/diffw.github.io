For a product, visual performance disconnected from the product's structure is meaningless.

Visual performance is intuitive, like an artist's 'graffiti,' not 'composition-creation';

Visual design combines reason and emotion, built on understanding and optimizing the structure.

Visual performance is about details and fragments;

Visual design is a comprehensive, complete, and logically and structurally clear solution.

Visual design solves problems through visual performance.

But first, you need to clarify what problems you're solving before starting design.

This requires product awareness and interaction awareness beyond visual performance.

If a visual designer can't fully understand the product, lacks product awareness, and has no sense of interaction, then honestly, they're not really a visual designer.
